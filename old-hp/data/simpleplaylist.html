simpleplaylist - simple playlist script for amarok

About

simpleplaylist is a small extension which dumps the played files with time in a file. So one can always know when you played which song. Release Notes

  • 0.3 Minor changes: e.g. fixed configuration dialog
  • 0.2 Save format changed to "DATE TITLE" instead of "TITLE DATE"
  • 0.1 The first release.

Installation

  1. Save it to your HDD.
  2. amarok -> Script Manager -> Install Script -> and choose the tar file

Usage

Just listen to music in amarok. The default behaviour is to save your listend tracks in /tmp/simp.
