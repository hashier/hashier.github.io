Official page of Christopher Loessl. Take a look around

github

You can find almost all of my projects on github.
On github you can always find the newest versions and code of (almost) all my projects. I listed some projects below for easier access but the links are just forwarding to github anyway (:

MacFolket

A Swedish - English dictionary for Mac OS X which is nicely integrated into the system.

idleSound

Menubar program that mutes your Mac if you haven't used it in quite some time or your monitor starts screensaver or locks itself.

ddate

For the geeks, a GPL ddate implementation for iOS. It also features a MIT implementation of the conversation library as an Objective-C category.

Blog (stackenblochen)

That's my blog. It's more or less a ranting blog. I usually get quite easily annoyed by hard- and software and I often think that people who created something never actually used/tried it out before they sold it.

Spotify-remote

Run this server on the machine that is running Spotify. Afterwards you can visit a website and control your Spotify through it. Right now only play/pause, next and prev is implemented and it looks quite ugly.

Spotify-link-to-song

Send easily the Spotify URL of the current playing track to someone you are talking to on Adium.
To install it directly from the Adium Xtra page click here.

Gimp CUDA Plug-In

GIMP Plug-In(s) wich are using CUDA for calculation.
This was before the GPU hype and I thought it would be nice to bring some GPU capabilities to GIMP so it had some features that Photoshop would lack.

SuperSAFT

If you have a supersaft server running anyone can send files to you by using the supersaft client. Advantage is that the sending user doesn't need to have an account on your machine.

flicp

With this tool you can send or get files from a fli4l router.

CBP Compiler

A compiler which can compile .cbp files. This was a university project at TU Braunschweig for the compiler building lecture and lab.
.cbp is a self defined language.
